### Finance-Management-Apps-Client
A modern, responsive frontend application to manage personal finance: track income, expenses, budgets, and visualize your financial data.

## Project Overview

This is the **client-side** of a Finance Management App. The core idea:

- Track **income** and **expenses**  
- Create and manage **budget categories**  
- Visualize financial data with interactive charts  
- Authenticate users (login/signup)  
- Provide a clean and user-friendly UI  
- Connect to a backend API for persistent data storage

  ## Tech Stack
- **React.js**
- **Tailwind CSS** (or CSS framework of your choice)
- **React Router** for routing 
- **Chart.js**for data visualization   
- **Fetch** for HTTP requests   
- **Context API for state management

  ## Features

- Add, edit, delete **transactions** (income / expense)  
- Create and manage **budget categories**  
- Filter transactions by date or category
- Visualize transactions in **charts** (bar, pie, line)  
- **User authentication** (login / register)  
- Responsive design ‚Äî mobile & desktop friendly

#Getting Started (Run Locally)
##Clone the repository:
git clone https://github.com/rabbiWD/Finance-Management-Apps-Client.git
cd Finance-Management-Apps-Client
npm install

## Create Environment File
Create a .env file in the root directory and add your environment variables:
VITE_apiKey=your_api_key
VITE_authDomain=your_auth_domain
VITE_projectId=your_project_id
VITE_storageBucket=your_storage_bucket
VITE_messagingSenderId=your_messaging_sender_id
VITE_appId=your_app_id

##Start the Development Server
npm run dev
